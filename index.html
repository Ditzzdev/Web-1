Topologi Jaringan

Topologi jaringan adalah cara menghubungkan perangkat dalam suatu jaringan.

Simulasi koneksi antar node

Arsitektur Jaringan

Arsitektur jaringan menggambarkan hubungan antara client dan server dalam jaringan.

💻 Client ⇄ 🖥️ Server

Pengalamatan

Pengalamatan adalah cara memberi alamat unik ke setiap perangkat dalam jaringan.

IP: 192.168.1.1 💡

Subnetting

Subnetting adalah membagi jaringan besar menjadi beberapa jaringan kecil.

📦 → 📦📦📦 (Pembagian jaringan)

CIDR

CIDR memungkinkan pengelompokan alamat IP dengan efisien.

192.168.0.0/24 ➜ 192.168.0.0/26

VLSM

VLSM adalah teknik subnetting dengan ukuran subnet yang bervariasi.

🧩 Subnet dinamis menyesuaikan kebutuhan
